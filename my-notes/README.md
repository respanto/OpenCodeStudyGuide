# My OpenCode Study Notes

Use this folder for your personal notes as you work through `OPENCODE_STUDY_GUIDE.md`.

## Suggested files

- `session-log.md` — date, what you tried, what worked
- `prompts-that-work.md` — copy prompts you want to reuse
- `mistakes.md` — things the agent got wrong (learn from them)
- `config-snippets.md` — your `opencode.json` and `AGENTS.md` patterns

Start with one file. Add more when you need them.
